# Mandates Workflow

## Status

Compatibility workflow surface under `Origination Match`.

## Current Role

`/mandates` still hosts the active mandate-definition flow used by `Origination Match`.

It remains operational, but it is no longer the official top-level product architecture by itself.

## Current Responsibility

Mandates currently define:

- target scope
- constraints
- search direction
- the launch point for research

## Rule

Developers should describe mandate work as part of the `Origination Match` solution module, not as a standalone permanent top-level module strategy.
